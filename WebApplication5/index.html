Perfect Numbers

In number theory, a perfect number is a positive integer that is e qual to the sum of its proper positive divisors, that is, the sum of its positive divisors excluding the number itself. The first perffect number is 6 -- its positive divisors (excluding itself) are 1,2, and 3, whose sum is 6.

This function will determine whether a given number is "perfect."


This function will display all Perfect Numbers from 1 - 10,000.

    function perfect() {
		var userInput = parseInt(document.getElementById("num1").value);
		var arry = [];

		}
		else {
		for (var i = 1; i < userInput; i++) {
			if (userInput % i === 0) {
				arry.push(i);

			var arrSum = 0;
				for (var j=0, len=arry.length; j < len; j++) {
					arrSum += arry[j];

	}

Palindrome Checker

A palindrome is a word, phrase, number, or sequence of characters which reads the same backward or forward. allowances may be made for adjustments to capital letters, puncuation, and word dividers. the word "palindrome was coined by the English playwright Ben Johnson in the 17th century from the Greek roots pali ("again") and dromos ("way, direction").

function palindrome() {
    var userInput = document.getElementById('pali1').value;
    var vali = validatePali(userInput);
	if (vali) {
		document.getElementById("showpali").innerHTML = "Please enter a word, phrase, or number.";
	} 
	else {
	var str = userInput.toUpperCase().replace(/\s+/g, '');
	strRev = str.split('').reverse().join('');
		if (str == strRev) {
			document.getElementById("showpali").innerHTML = "Yes, " + userInput + " is a Palindrome!";
		}
		else {
			document.getElementById("showpali").innerHTML = "No, " + userInput + " is not a Palindrome!";
		}
	}
}

Factorial!

In mathematics, the factorial of a non-negative integer n, denoted by n!, is the product of all positive integers less than or equal to n. For example, 5! = 5 x 4 x 3 x 2 x 1 = 120. This program calculates the factorial of a given number.

function factorial() {
var userInput = parseInt(document.getElementById("num1").value);
var vali = validate(userInput);
var posi = positive(userInput);
if (!vali) {
	document.getElementById("show").innerHTML = "Please enter a number!";
}
else if (!posi) {
	document.getElementById("show").innerHTML = "Nice try.  Go read a paper on negative factorials.  In the meantime, please enter a positive integer.";
}
else {
	var fac = 1
	for (i = 1; i<=userInput; i++) {
		fac = fac * i;
	}
	document.getElementById("show").innerHTML = fac;
}
}

Fizz Buzz

This program accepts two numbers between 1 and 100 from the user. The program then iterates through all numbers from 1 to 100, printing numbers in ascending order to the screen. For each number that is a multiple of the first number input by the user, the program will print "Fizz" instead of the number. For each number that is a multpiple of the second number input by the user, the program will print "buzz" instead of the number. For numbers that are multiples of both numbers, the program will print "FizzBuzz" instead of the number.
