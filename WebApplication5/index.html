Prime!

A prime number is a natural number greater than 1 that has no positive divisors other than 1 and itself. A natural number greater than 1 that is not a prime number is called a composite number. For example, 5 is prime because 1 and 5 are its only positive integer factors, whereas 6 is composite because it has the divisors 2 and 3 in addition to 1 and 6. This program will display all prime numbers up and, if applicable, including the number entered by the user, while excluding composite numbers.

function primeNo() {
var userInput = parseInt(document.getElementById("prime1").value);
	if  {
	var arr = [];
	for (a=2; a<=userInput; a++) {
		var prime = true;
		
		for (b=2; b < a; b++) {
			if(a % b === 0)
			{
				prime = false;
			}
		}
		if (prime) {
			arr.push(a);
		}
	}
		document.getElementById("printprime").innerHTML = arr.join(" ");
	}	
}

Perfect Numbers

In number theory, a Perfect Number is a positive integer that is equal to the sum of its proper positive divisors, that is, the sum of its positive divisors excluding the number itself. the first perffect number is 6 -- its positive divisors (excluding itself) are 1,2, and 3, whose sum is 6.

This function will determine whether a given number is "Perfect."


This function will display all Perfect Numbers from 1 - 10,000.

Palindrome Checker

A palindrome is a word, phrase, number, or sequence of characters which reads the same backward or forward. allowances may be made for adjustments to capital letters, puncuation, and word dividers. the word "palindrome was coined by the English playwright Ben Johnson in the 17th century from the Greek roots pali ("again") and dromos ("way, direction").

function palindrome() {
    var userInput = document.getElementById('pali1').value;
	if  {
	var str = userInput.toUpperCase().replace(/\s+/g, '');
	strRev = str.split('').reverse().join('');
		if (str == strRev) {
			document.getElementById("showpali").innerHTML = 
            "Yes, " + userInput + " is a Palindrome!";
		}
		else {
			document.getElementById("showpali").innerHTML = 
            "No, " + userInput + " is not a Palindrome!";
		}
	}
}

Factorial!

In mathematics, the factorial of a non-negative integer n, denoted by n!, is the product of all positive integers less than or equal to n. For example, 5! = 5 x 4 x 3 x 2 x 1 = 120. This program calculates the factorial of a given number.

function factorial() {
var userInput = parseInt(document.getElementById("num1").value);
    if {
	    var fac = 1
	    for (i = 1; i<=userInput; i++) {
		    fac = fac * i;
	    }
	    document.getElementById("show").innerHTML = fac;
    }
}

Fizz Buzz

This program accepts two numbers between 1 and 100 from the user. The program then iterates through all numbers from 1 to 100, printing numbers in ascending order to the screen. For each number that is a multiple of the first number input by the user, the program will print "Fizz" instead of the number. For each number that is a multpiple of the second number input by the user, the program will print "buzz" instead of the number. For numbers that are multiples of both numbers, the program will print "FizzBuzz" instead of the number.



function fizzBuzz() {
var user1 = parseInt(document.getElementById('fb1').value);
var user2 = parseInt(document.getElementById('fb2').value);
if {
		var numArr = [user1, user2];
		var fizzRemain = 1;
		var buzzRemain = 1;
		var fizzBuzzRemain = 1;
		var fizz = numArr[0];
		var buzz = numArr[1];
		var fizzBuzz = fizz * buzz;
		var output = "";
		output += "";
			for (var i = 1; i <= 100; i++) {
				numArr[i] = parseInt(numArr[i]);
				fizzRemain = i % fizz;
				buzzRemain = i % buzz;
				fizzBuzzRemain = i % fizzBuzz;
				if (fizzBuzzRemain == 0) {
					output += "FizzBuzz ";
				}
				else if (fizzRemain == 0) {
					output += "Fizz ";
				}
				else if (buzzRemain == 0) {
					output += "Buzz ";
				}
				else {
					output += i + " ";
				}
				
			}
		document.getElementById("showfb").innerHTML = output;
	}	
}

Happy Numbers

A Happy Number is a number defined by the following process: Starting with any positive integer, replace the number by the sum of the squares of its digits, and repeat the process until the number either equals 1 (where it will stay), or it loops endlessly in a cycle which does not include 1. Those numbers for which this process ends in 1 are happy numbers, while those that do not end in 1 are unhappy or sad numbers. This program is designed to display the first 5 Happy Numbers beginning with 1.
